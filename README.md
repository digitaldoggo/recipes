# Recipes

My favorite cooking and baking recipes, served as a static site.

**Live site**: [digitaldoggo.github.io/recipes](https://digitaldoggo.github.io/recipes/)

## Running locally

```bash
yarn start   # dev server on port 3000
yarn build   # static site output
yarn deploy  # push to GitHub Pages
```

Requires Node.js and [Docsify](https://docsify.js.org/).
